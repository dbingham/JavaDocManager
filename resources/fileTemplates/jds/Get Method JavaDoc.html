This is a template used for generating the JavaDoc declaration for a getter method each time it is generated by the program, e.g. when choosing the Generate JavaDoc based on method signature quick fix of the JavaDoc Sync plugin.
Predefined variables will take the following values:
${NAME}   name of the getter method for which the JavaDoc is generated.
${FIELD_NAME}   the class attribute on which this getter method operates.
${FIELD_COMMENT}   the doc comment for the class attribute on which this getter method operates.
${CLASS}   the class that this getter method is part of.
${RETURN_TYPE}   the return type of this getter method.
