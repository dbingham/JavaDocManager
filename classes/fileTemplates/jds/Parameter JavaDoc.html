This is a template used for generating the JavaDoc declaration for a method each time it is generated by the program, e.g. when choosing the Generate JavaDoc based on method signature quick fix of the JavaDoc Sync plugin.

Note: This is an inline template which is not directly used but included by other templates that can possibly contain parameters.
Predefined variables will take the following values:
${PARAM}   name of the method parameter for which the JavaDoc is generated.
${TYPE}   data type of the method parameter for which the JavaDoc is generated.
